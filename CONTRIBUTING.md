# Contributing

## Setup

```sh
git config core.hooksPath .githooks   # auto-runs rustfmt on commit
sh scripts/fetch-official-fixtures.sh # test fixtures (--large adds the 21K-event log)
```

Tests that depend on the fetched fixtures skip themselves when the files are
absent, so a plain checkout still passes.

## Quality gates

All of these must pass before a PR:

```sh
cargo fmt --check
cargo clippy --all-targets -- -D warnings
cargo test
cargo deny check
```

## Conventions

- Tests first where practical; every I/O change needs a round-trip test.
- Conventional commits in English (`feat:`, `fix:`, `refactor:`, `test:`, `docs:`).
- Branch names: `feat/xxx`, `fix/xxx`, `refactor/xxx`, `docs/xxx`.
- Keep dependencies minimal (`cargo machete` should stay clean).
- `unsafe` is forbidden in `ocel-core` / `ocel-cli` (workspace lint);
  `ocel-py` allows only the glue generated by pyo3 macros.

## Python bindings

```sh
cd crates/ocel-py
maturin develop   # or: maturin build --release && pip install ../../target/wheels/ocel-*.whl
```
